Trove Help

Use the Trove button in the top left to return to the main Trove menu at any time.

Browsing Recordings

Trove has been designed to be intuitive. Browse artists by clicking on their name. Select a date by clicking on an artist, a year, then the date. If there is only one recording for a date you will be taken to the recording detail page.

There may be multiple recordings for a single date. Every recording for a date will be listed together. The recordings are listed by their Archive Identifier. There may be an orange box below an identifier. This box lists the source summary information from etreedb.org.

Favorites

The recording detail page lists information about the artist, date, venue, and there is a Show Description button which will toggle display of the source text file listing recording info, set list, and more.

There are two icons on the recording detail page. Clicking the icon will toggle the recording as a favorite for your device. You can then use the Fav. Recordings button at the top to browse all recordings you have marked as favorite.

Clicking the icon will set a marker that you have listened to the recording and when you listened to it. When browsing a date with multiple recordings the icon will be on any recording you have already listened to.

Bookmark

You can book one and only one recording at a time. When a bookmark exists a icon appears next to the help button at the very top. Clicking this will take you back to the bookmarked recording.
